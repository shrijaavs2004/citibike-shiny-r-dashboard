# Shiny Dashboard – Individual Assignment

This repository contains a Shiny application developed for the course assignment.

## Features
- Interactive input slider
- Reactive dataframe
- Histogram visualization
- Summary statistics table

## Installation

### R

Install dependencies:

renv::restore()

Run the app:

shiny::runApp()
